OpenDocument Text Exporter Manual

This manual is for Org-7.8.08’s OpenDocument Text Exporter Manual.

Orgmode1 supports export to OpenDocument Text (ODT) format using the ‘org-odt.el’ module. Documents created by this exporter use the OpenDocument-v1.2 specification2 and are compatible with LibreOffice 3.4.

Footnotes

Versions 7.8 or later

(2)
